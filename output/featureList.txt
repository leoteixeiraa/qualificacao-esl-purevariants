
Nome Visível:	WeatherMonitor
Nome Único:	WeatherMonitor

Descrição:

Estações meteorológicas são usadas para adquirir dados sobre as condições climáticas atuais; esses dados são usados para
fazer previsões meteorológicas precisas. Estações meteorológicas também são importantes para o estudo do clima e do tempo.

Este é o recurso básico do nosso exemplo de estação meteorológica. Ele fornece toda a funcionalidade básica necessária.
Esta estação possui três recursos principais, sendo um deles opcional. Assim, toda estação meteorológica tem pelo menos
um sensor para medir dados meteorológicos e pelo menos uma forma de se comunicar com um usuário ou um PC.

O aplicativo é capaz de medir velocidade do vento, temperatura ou pressão do ar. O aplicativo pode
mostrar os valores medidos em um LCD ou transferi-los para um PC.

---------------------------------------------------------------------------------------------------------


Nome Visível:	Saída
Nome Único:	OutputSupport

Descrição:

Saída dos dados de medição. Fornece a capacidade de passar os valores medidos para o usuário ou outros componentes do sistema.

---------------------------------------------------------------------------------------------------------


Nome Visível:	LCD
Nome Único:	Display

Descrição:

Este recurso fornece suporte para um display LCD embutido na estação meteorológica.

Como a economia de energia é importante em estações meteorológicas, que precisam operar por longos períodos
sem conexão elétrica, os displays são muito simples. Apenas 60 caracteres em 3 linhas podem ser exibidos.

E os displays embutidos podem usar algumas técnicas de economia de energia, como desligar o display quando
não está em uso ou diminuir a luz de fundo se houver luz suficiente.

---------------------------------------------------------------------------------------------------------


Nome Visível:	Transferência de Dados para PC
Nome Único:	PCConnection

Descrição:

Este recurso fornece a capacidade de comunicação. Usar este recurso permite conectar
a um PC e controlar toda a estação meteorológica a partir de um PC.

Dois tipos de conexão podem ser usados: USB ou RS232. Ambas as interfaces podem ser usadas para conectar a estação meteorológica
diretamente a um PC.

Mas esses recursos são alternativos, não podem ser usados juntos.

---------------------------------------------------------------------------------------------------------


Nome Visível:	USB
Nome Único:	USBLine

Descrição:

Um conector USB. Este recurso possibilita o uso de uma interface USB. Isso pode ser usado para conectar
diretamente a estação meteorológica a um PC. Para usar essa interface, é necessário um protocolo de transporte serial.

O protocolo não faz parte deste recurso. O recurso "Protocolo" é responsável pelos protocolos de transporte.

A interface USB é muito mais rápida que a interface RS232, podendo atingir cerca de 12 Mbit/s. Mas o comprimento máximo
do cabo USB é de 5m, então é muito mais curto que o cabo da interface RS232.

---------------------------------------------------------------------------------------------------------


Nome Visível:	Protocolo
Nome Único:	Protocol

Descrição:

A transferência de dados utiliza um protocolo especial. A transferência de dados precisa ser segura. Esta estação meteorológica é capaz
de enviar um fluxo constante de dados, para observação meteorológica em tempo real.

Duas alternativas diferentes podem ser usadas. Você pode usar um protocolo próprio, que não é usado por nenhuma outra
estação meteorológica, ou pode usar um protocolo USP padrão, que é usado sobre SLIP.

Os detalhes são explicados nos respectivos recursos.

---------------------------------------------------------------------------------------------------------


Nome Visível:	UDPoverSLIPProto
Nome Único:	UDPoverSLIPProto

Descrição:

Este recurso fornece um protocolo de comunicação padrão. Ele utiliza o protocolo padrão User Datagram Protocol (UDP).

Este é um protocolo muito simples. Esse protocolo é usado sobre o protocolo Serial Line Internet Protocol (SLIP), que também é
muito simples. Mas não é muito seguro. O UDP usa soma de verificação para calcular se uma mensagem está correta,
mas não pode corrigir a mensagem se ela estiver corrompida. Essa mensagem precisa ser reenviada. E se
uma mensagem for perdida, ninguém perceberá, pois o SLIP não tem detecção de erro.

UDP e SLIP têm apenas uma pequena sobrecarga para endereçamento.

Nome Visível:	Sensores
Nome Único:	Sensors

Descrição:

Este recurso fornece funcionalidade básica para conectar sensores à estação meteorológica.

Três tipos diferentes de sensores podem ser conectados à estação meteorológica.

---------------------------------------------------------------------------------------------------------


Nome Visível:	Pressão
Nome Único:	AirPressureSensor

Descrição:

Este recurso adiciona suporte para um sensor de pressão do ar. Qualquer sensor para medir a pressão do ar pode
ser usado com a estação meteorológica.

O intervalo de medição pode ser escolhido entre 1 segundo e uma vez por semana.

---------------------------------------------------------------------------------------------------------


Nome Visível:	Temperatura
Nome Único:	TemperatureSensor

Descrição:

Este recurso permite conectar sensores de temperatura à sua estação meteorológica. A estação meteorológica
consegue capturar temperaturas entre -40°C e +100°C corretamente. Qualquer sensor de temperatura digital
pode ser usado.

O intervalo de medição pode ser escolhido entre 1 segundo e uma vez por semana.
